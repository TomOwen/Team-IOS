How to Add a New Study or Clinical Trial

To add or change a Study, click on the "Studies" icon on the bottom TAB bar.



To add a new Study click on the "+" button in the Top Toolbar.



Study Number:

Enter a study identifier such as NCT12345.

Company/Pharma:

Enter the sponsor name such as Bristol Myers or "Internal-100" etc.

Short Description:

Enter a short description of the trial or study.

Info File Name:

This field is optional. If you enter a file name, then it will be used by the "Study Detail" screen (accessed from the "Studies" screen when you select a study) to allow you to view that file. Normally this file will be either a Word document (.doc) or a PDF (.pdf).

Enter a filename such as bms12345.doc

When you go to the "Study Detail" screen you will see a "View Study Document" button allowing you to view the current document.

Partial Response % Decrease

If the study is using the standard RECIST critieria for determining response, you can leave the default value of 30. Any lesion that has a size (longest diameter) decrease of 30% or more will be classified as a Partial Response.

Progressive Disease % Increase

If the study is using the standard RECIST critieria for determining response, you can leave the default value of 20. Any lesion that has a size (longest diameter) increase of 20% or more will be classified as a Progressive Disease.

Complete Response will always be a 100% decrease in size and Stable Disease will always be defined as lesions that do not fall between Partial Response and Progressive Disease.
Saving the New Study

Remember to click the "Save" button on the top right Tool Bar to save the study.
